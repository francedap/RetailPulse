istruzioni:

Entri nella cartella: cd RetailPulse
Creare l'ambiente virtuale: python -m venv venv
Entrare nell'ambiente virtuale: 
    Windows: venv\Scripts\activate
    Mac/Linux: source venv/bin/activate
Installare le Librerie: pip install -r requirements.txt

per far partire il ""server"": streamlit run app.py

agiornare il requirements.txt delle librerie: pip freeze > requirements.txt

per popolare il DB:
    python populate_db.py
